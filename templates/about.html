FAQ

What is a security header?
Security headers are HTTP response headers that, when set, can enhance the security of your website by helping to mitigate attacks and secure communications.

Why is SSL/TLS important?
SSL/TLS certificates encrypt the data exchanged between a user's browser and a website, ensuring that sensitive information like passwords and credit card details are transferred securely.

What is robots.txt?
Robots.txt is a file webmasters use to instruct web crawlers about which pages on their site should or shouldn't be crawled and indexed for search engine results.

How do I interpret the results provided by this site?
Each analysis result comes with a detailed description that explains what the data means and why it's important for the security and functionality of the website.

What should I do if no SSL/TLS certificate is found for a site?
A missing SSL/TLS certificate indicates that the site is not securing its data, which could pose a risk if sensitive information is exchanged. It's advisable to exercise caution and avoid sharing personal information on such sites.

What does the security.txt file tell us?
The security.txt file provides information about how to report security issues to the website owner, and may include guidelines for security researchers.
